# ibcheck.sh README

## Description
ibcheck.sh is a script that performs various checks on InfiniBand (IB) network configurations.

## Prerequisites
- InfiniBand hardware and drivers installed
- Bash shell

## Usage

1. Make the script executable:
    ```bash
    chmod +x ibcheck.sh
    ```

2. Run the script:
    ```bash
    ./ibcheck.sh
    ```

## Features
- Check IB link status
